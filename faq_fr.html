    Par défaut, Windows bloque l'installation de toute application qui ne provient pas du Microsoft Store.

  1. Sélectionnez Démarrer  > Paramètres  > Applications  > Paramètres avancés des applications .

  2. Sur la première option Choisir où obtenir les applications, choisissez N'importe où.

    Autre possibilité, le contrôle intelligent des applications est activé.

  1. Sélectionnez Démarrer  > Sécurité Windows  > Controle des applications et du navigateur  >

  2. Sous la première option Contrôle intelligent des applications , cliquez sur Paramètres de contrôle intelligent des applications.

  3. Choisissez Désactivé.

Enfin, il se peut que ce soit votre pare-feu ou votre antivirus qui bloque l'installation. Essayez de les désactiver temporairement.

Il est possible de jouer à POKéMON BRDB sur d'autres systèmes que Windows en utilisant des applications comme Wine ou CrossOver.

  1. Installez l'un des deux logiciels.

  2. Faites un clic droit sur le fichier .exe d'installation du jeu POKéMON BRDB.

  3. Cliquez sur l'option Ouvrir avec et choisissez Wine ou CrossOver.

  4. Toujours avec Wine ou CrossOver, ouvrez le fichier game.exe qui se trouve dans le dossier où vous avez installé le jeu (par défaut C:\Program Files (x86)\POKéMON ROUGE ECLATANT & BLEU PROFOND).

Il est possible de jouer à POKéMON BRDB sur Android en utilisant l'application JoiPlay (à ce jour, il n'existe pas de méthodes pour jouer sur IOS).

  1. Sur le site de JoiPlay, téléchargez les fichiers .apk de JoiPlay et de RPG Maker Plugin.

  2. Installez l'apk de JoiPlay sur votre téléphone Android (par défaut JoiPlay-x.xx.xxx-patreon-release.apk).

  3. Téléchargez le dossier du jeu POKéMON BRDB sur votre téléphone Android et placez-le à l'endroit que vous désirez.

  4. Sur l'application JoiPlay > Le signe  en haut à droite > Add game > Choose > Sélectionnez le fichier game.exe dans le dossier du jeu que vous avez précédemment mis sur votre téléphone.

  5. Suivez les indications et remplissez le nom et la version que vous désirez pour l'application (cela n'a pas d'importance).

  6. Vous pouvez également choisir que vous désirez pour l'icone de l'application. Une icone par défaut est présente dans le dossier du jeu.

  7. JoiPlay va vous proposer d'extraire les fichiers encryptés. Choisissez Extract.

  8. JoiPlay va vous demander de téléchargez un fichier RTP. Téléchargez-le en cliquant sur Download puis sur la page internet qui s'est ouverte, scrollez et choisissez RPG Maker XP > Download Run Time Package.

  9. Sur JoiPlay, appuyez maintenant sur Choose et sélectionnez le fichier RTP que vous avez téléchargez à l'étape précédente.

  10. Quittez JoiPlay et installez l'autre fichier .apk que vous avez téléchargez (par défaut RPGMPlugin-x.xx.xx-patreon-release.apk).

  11. Retournez sur JoiPlay, cliquez sur l'icone du jeu et acceptez les autorisations nécessaires.

Dans certains cas, il est possible que les visuels sur mobile ne soient pas bons. Essayez de suivre les étapes suivantes pour corriger le rendu :

  1. Mettez à jour le plugin RPG Maker de JoiPlay en le téléchargeant via le lien suivant : lien

  2. Activez l'option Optimize Map dans les options de JoiPlay.

Le jeu prend en charge les manettes. Les touches par défaut sur une manette XBox sont :

X > Touche action (valider/confirmer)

B > Touche retour (annuler/effacer)

Le stick analogique gauche > Déplacements

Que ce soit celles du clavier ou de la manette, il est possible de configurer les touches en appuyant sur F1 en cours de partie.

Dans la fenêtre qui s'ouvre, vous pouvez modifier les touches du clavier en allant sur l'onglet Keyboard et celles de la manette en allant sur l'onglet Gamepad

Les touches par défaut sont les suivantes :

Nom dans menu Manette (Nom dans menu) Clavier Action
A X (Button 1) Shift, Z -
B B (Button 2) Esc, Num 0, X Cancel, Menu
C A (Button 3) Space, Enter, C Action Button
X Y (Button 4) A -
Y LB (Button 5) S -
Z RB (Button 6) D -
L Back (Button 7) Q, Page Up Previous Page
R Start (Button 8) W, Page Down Next Page

Pour toutes les questions concernant le fonctionnement du jeu, rendez-vous sur le Wiki du jeu.

Vous trouverez plein d'infos utiles comme les évolutions modifiées, les différents terrains, etc.

Le jeu peut contenir jusqu'à 3 sauvegardes indépendantes. Si jamais vous souhaitiez commencer une 4ème partie, il vous sera proposé de supprimer l'une des 3 autres.

Il est également possible de supprimer manuellement les fichiers de sauvegarde mais faites toujours bien attention à possèder des sauvegardes avec des numéros qui se suivent et qui commencent par 1. Vous pouvez renommer vos fichiers de sauvegarde pour changer leur numéro si nécessaire.

Par défaut, les fichiers de sauvegarde se nomment saveX.rxdata et se trouvent dans le dossier C:\Users\xxx\AppData\Local\VirtualStore\Program Files (x86)\POKéMON ROUGE ECLATANT & BLEU PROFOND.
